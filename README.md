# React Firebase App

![React Firebase App](/src/images/React-Firebase.gif)

## Install

Some basic Git commands are:

```
$ git clone https://github.com/mohamedsamara/React-Firebase-App.git
$ cd project
$ npm install

```

## Start

npm start

## Simple build for production

npm run build
